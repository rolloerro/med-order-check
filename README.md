# 🩺 MED-ORDER-CHECK

**Medical Order Validation Service**  
Проверка корректности медицинских назначений (MVP)

---

## 🧠 Идея проекта

**MED-ORDER-CHECK** — это backend-сервис для автоматической проверки медицинских назначений.  
Он не заменяет врача, а **подсвечивает потенциальные риски и ошибки** до того, как они станут проблемой.

> **Медицина = сервисы**  
> **Боты = интерфейсы**

Этот проект — именно сервис.

---

## 🚀 Что делает сервис

На текущем этапе (MVP):

- принимает структуру назначения (препарат, доза, путь введения и т.д.)
- валидирует данные по формальным правилам
- подготавливает архитектуру для:
  - клинических правил
  - справочников
  - AI / RAG-модулей

---

## 🔍 Примеры проверок (план развития)

- ❗ некорректные дозировки  
- ❗ несовместимые пути введения  
- ❗ возрастные ограничения  
- ❗ потенциально опасные назначения  
- ⚠️ предупреждения и рекомендации  

---

## 🧩 Где применяется

- телемедицина  
- клинические формы и выписки  
- EMR / HIS системы  
- медицинские боты (как UI)
